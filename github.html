⏰ 30-Minute Daily Routine

Ye follow karo har din — guaranteed green square! 🟩

00:00 – 05:00 min

  • GitHub profile open karo

  • Aaj ka plan decide karo
05:00 – 20:00 min
Main Work
  • Current project mein improve karo
  • Feature add ya bug fix karo
  • Descriptive commit message likho
20:00 – 25:00 min
PR Workflow
  • New branch pe kaam karo
  • PR banao → merge karo
  • Branch delete karo (clean!)
25:00 – 30:00 min
Documentation
  • README mein koi update karo
  • Screenshots add karo project ke
  • Final commit — done! ✅
